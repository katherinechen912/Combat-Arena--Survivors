Combat Arena: Survivors

Hey, this is a 2D Survivors-like action game I put together using Godot 4. It's a fast-paced arena survival game where your only goal is to outlast a massive pixel swarm.

How to play
You just use WASD or the arrow keys to move. You don't need to click to shoot—your character auto-attacks the closest enemies, so you can just focus on dodging and not getting cornered. When enemies drop blue EXP vials, grab them to level up and pick your stat upgrades. 

Dev notes
I'm making the source code available here if anyone wants to poke around a Godot 4 project. A few things I focused on under the hood:

- Built a custom exponential XP curve so the late-game stays challenging.
- Set up a state machine for smooth animation transitions.
- Designed a custom UI and HUD with a retro-cyberpunk neon color palette.

Feel free to grab the code, fork it, or just play the game. Thanks for checking it out!
